# nut-slot-calculator
# Guitar Nut Slot Spacing Calculator

ギター／ベース用ナットの弦間隔を計算し、  
**印刷してそのまま加工に使えるテンプレート**を生成するツールです。

---

## 概要

- 弦ゲージを考慮した **実効中心割** 計算
- 6弦 / 7弦 / 4弦ベース / 5弦ベース対応
- 印刷用テンプレート（100mm検尺付き）
- ブラウザ完結

---

## 使用方法

1. htlmファイルを実行
2. ナット幅を入力
3. 各弦のゲージを入力
4. 「計算」ボタンを押す
5. 表示されたテンプレートを **印刷倍率100%** で印刷
6. STLにて治具やナットとして出力可能

※ 印刷後、必ず **100mm検尺** を実測で確認してください。

---

## 対応環境

- Google Chrome（推奨）

※ スマートフォンでは使用を想定していません。

---

## 注意事項

- 印刷時は **ページの拡大縮小を無効** にしてください
- 実加工前に必ず実測で確認してください
- 本ツール使用による加工ミス等の責任は負いかねます
